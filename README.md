# yue-playground

```sh
$ yarn install
$ git submodule update --init

$ wget https://github.com/yue/yode/releases/download/v0.3.4/yode-v0.3.4-linux-x64.zip
$ unzip *.zip
$ chmod +x yode

$ ./yode yue-sample-apps/editor/js/editor.js
$ ./yode src/browser
```
